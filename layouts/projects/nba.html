NBA % of Team Points Calculator

Ademide Kosoko
JavaScript · Tailwind · 2024

As a basketball fan, this application allows players to view stats of NBA players and calculate the percentage of points they contribute to their team.

View on GitHub Use Project

Features of this project:

  1. Create Reviews of Albums
  2. Rate Albums out of 5 stars
  3. Allows users to create accounts and store their data
  4. Can filter reviews by stars
  5. Utilizes Spotify's API to get tracks for search
