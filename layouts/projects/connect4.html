Connect 4

Ademide Kosoko
JavaScript · Game · 2024

This project simulates a game of Connect 4 with Player 1 being the Red Player and Player 2 being the Yellow Player. The game can be played against a friend or against an AI. The game also allows you to undo previous moves.

View on GitHub Play Game

Features of this project:

  1. Play against a friend
  2. Play against an AI
  3. Undo previous moves
  4. Animate pieces as you place them
  5. Determines winner using sliding window logic
